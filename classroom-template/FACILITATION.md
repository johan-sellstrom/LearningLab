# Facilitation Guide (Learning Lab Best Practices Alignment)

This guide turns the RSAC Learning Lab best practices into concrete actions for this course. Use it alongside `OUTLINE.md` and the per-lab guides.

## Core facilitation principles
- You are a facilitator, not a presenter.
- Target time split: 25-30% brief instruction, 70-75% exercises.
- Never talk for more than 15 minutes without an interaction.

## Timing map (2 hours)
- 0:00-0:05 Welcome + goals (light framing, no deep lecture)
- 0:05-0:12 Warm-up exercise (low-stakes, table-based)
- 0:12-1:40 Labs 00-05 (timeboxed; each lab includes: 3-5 min setup, 10-20 min hands-on, 3-5 min report-out)
- 1:40-1:55 Whole-room synthesis + Q&A
- 1:55-2:00 Wrap + next steps

## First exercise (early, low-stakes)
Purpose: build comfort and participation before harder content.
Example (5-7 min):
- Table prompt: "Pick one privacy risk you have seen in identity systems and one mitigation you are curious about."
- Each table picks a scribe and a reporter.
- 30-second report-out per table.

## Table roles (rotate each lab)
- Facilitator: keeps the group on task.
- Scribe: captures key points on flipchart.
- Reporter: shares highlights to the room.
- Timekeeper: watches the lab timer.

## Room flow and movement
- Use tables of six; give each table a flipchart prompt.
- Mix people between tables once (after Lab 02) to increase networking.
- Use at least one whole-room share-out per 2 labs.

## Pacing and engagement
- Start each lab with a 2-3 minute "why this matters" frame.
- Add a checkpoint question 8-10 minutes in.
- End each lab with a concrete outcome check (endpoint, proof, or UI cue).
- If a lab runs long, shorten the report-out, not the exercise.

## Reading the room
- Watch for dominant voices; invite quieter participants explicitly.
- Circulate, ask "What is blocking you?" rather than "Do you have questions?"
- If multiple tables are stuck on the same issue, pause and do a 2-minute reset for the whole room.

## Setup and logistics
- Arrive 15-20 minutes early.
- Confirm projector, whiteboard, and flipcharts are ready.
- Write table labels or prompts on flipcharts before attendees arrive.
- Greet attendees and guide them to mixed tables if needed.

## Whole-room prompts (use 2-3 times)
- "What surprised you in this lab?"
- "Which privacy property did we improve and why?"
- "What trade-off did you notice?"

## Success criteria for facilitation
- Participants are talking more than the facilitator.
- Every table reports out at least twice.
- Each lab ends with a tangible artifact (endpoint response, proof, status check).
